# Theme Cycling Feature

The Terminal Commander now supports 4 beautiful color themes that can be cycled using the 't' or 'T' key.

## Available Themes

### 1. Dark Theme (Default)
- Black background (#000000)
- White foreground (#FFFFFF)
- Blue active headers
- Dark cyan selection highlight
- Perfect for low-light environments

### 2. Light Theme
- White background (#FFFFFF)
- Black foreground (#000000)
- Blue active headers
- Sky blue selection highlight
- Ideal for daytime use

### 3. Solarized Dark
- Base03 background (#002b36)
- Base0 foreground (#839496)
- Solarized blue headers
- Solarized cyan selection
- Easy on the eyes with scientifically-designed color palette

### 4. Solarized Light
- Base3 background (#fdf6e3)
- Base00 foreground (#657b83)
- Solarized blue headers
- Solarized cyan selection
- Light variant with excellent contrast

## How to Use

1. Press 't' or 'T' to cycle to the next theme
2. The theme changes instantly across all UI elements
3. A status message shows the new theme name
4. Themes wrap around (after the last theme, returns to the first)

## Theme Application

Themes apply to all UI components:
- File browser panes (headers, selections, file listings)
- Status bar
- Text editor (including line numbers)
- Diff viewer (with color-coded differences)
- Search results
- Hash algorithm selector
- Archive format selector
- Help screen

All colors are carefully chosen to ensure:
- Good contrast and readability
- Consistent visual hierarchy
- Color-blind friendly options
- Reduced eye strain
